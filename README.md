# ShoppingCartProj
Early e-commerce website project, uses ASP.NET Core MVC, Entity Framework Core, and SQL Server. Supports shopping cart and product filtering using AJAX.
MagiStore - E-Commerce Shopping Cart Project
This project demonstrates a complete shopping flow from product browsing to session-based cart management and database-driven order processing.

🚀 Features
Dynamic Product Catalog: Displays products fetched directly from SQL Server.
AJAX-Powered Shopping Cart: Add items to the cart instantly without refreshing the page using jQuery AJAX.
Session Management: Stores cart data temporarily in Server-Side Session (JSON serialized) for guest users.
Smooth Checkout Process: Collects customer information and persists orders into the database.
Master-Detail Data Model: Saves orders in a structured Orders and OrderDetails relationship.
Code-First Approach: Database schema managed entirely via Entity Framework Core Migrations.

🛠️ Technologies Used
Backend: ASP.NET Core MVC (.NET 8.0)
Database: SQL Server, Entity Framework Core
Frontend: HTML5, CSS3, Bootstrap 5, JavaScript (jQuery/AJAX)
Tools: Visual Studio, Git/GitHub
